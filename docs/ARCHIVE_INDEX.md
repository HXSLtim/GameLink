## 💡 使用说明

### 查找文档
1. 核心文档: 查看 `/docs/` 下对应目录
2. 历史报告: 查看 `/archive/` 对应位置
3. API 文档: 查看 `/docs/api/` 或 `/backend/docs/swagger.yaml`

### 添加新文档
1. 新文档放在对应功能目录
2. 更新 `/docs/INDEX.md` 
3. 记录到本文档索引
